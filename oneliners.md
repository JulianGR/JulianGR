##### Terminal

###### Check the checksum MD5/ SHA256  of a file in Windows
````bash
certutil -hashfile "path\to\file.exe" md5(/sha256)
````

##### Git

###### Force a push and delete all changes
````bash
git reset --hard origin/master
````
